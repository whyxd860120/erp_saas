# 🚀 菜单管理 - 完整使用指南

## ⚠️ 当前问题

前端显示 **404 错误**：
```
GET http://localhost:5173/api/v1/menus 404 (Not Found)
```

**原因**：后端服务器未启动

---

## ✅ 立即解决

### 步骤1：启动后端服务器

**双击运行**：`启动后端服务器.bat`

将会看到类似输出：
```
🚀 Server is running on port 3000
📝 API Documentation: http://localhost:3000/api-docs
✅ Database connected
开始初始化权限数据...
权限数据初始化完成
```

### 步骤2：刷新前端页面

在浏览器中访问：`http://localhost:5173/menus`

按 **F5** 或 **Ctrl+R** 刷新页面

### 步骤3：看到菜单

现在应该能看到所有菜单了！🎉

---

## ⚠️ 常见问题

### Q: 双击后端脚本没反应？
**A**: 
1. 右键脚本选择"以管理员身份运行"
2. 或者手动打开终端运行：`cd e:\project\ERP2026\inventory-system && npm run dev`

### Q: 端口3000被占用？
**A**:
1. 关闭占用3000端口的程序
2. 或修改 `.env` 文件中的 `PORT=3001`

### Q: 数据库连接失败？
**A**:
1. 检查MySQL服务是否运行
2. 检查 `.env` 中的 `DATABASE_URL`
