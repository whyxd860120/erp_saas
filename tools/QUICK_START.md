### 安全注意事项
- ⚠️ 还原会覆盖目标数据库的所有数据
- ⚠️ 建议在空数据库或新数据库中还原
- ⚠️ 操作前确认备份文件完整性
- ⚠️ 限制备份文件的访问权限

## ❓ 常见问题

### Q: 提示"找不到 MySQL 工具"？
**A**: 确保 MySQL 已安装。工具会自动搜索常见路径，包括：
- C:\tools\mysql\current\bin ✓ (当前使用)
- C:\mysql\bin
- C:\Program Files\MySQL\MySQL Server 8.0\bin

### Q: 备份失败（退出码 2）？
**A**: 检查以下几点：
1. MySQL 服务是否运行: `Get-Service MySQL`
2. 数据库配置是否正确
3. 用户名密码是否正确
4. 备份目录是否有写入权限

### Q: 还原时出现唯一键冲突？
**A**: 这是因为目标数据库已有数据。解决方法：
1. 还原到空数据库
2. 或先清空目标数据库
3. 或创建新数据库进行还原

### Q: 中文显示乱码？
**A**: 工具已自动使用 utf8mb4 字符集。如果仍有问题：
1. 确保数据库使用 utf8mb4 字符集
2. 检查备份文件编码是否为 UTF-8
3. 重新备份并测试
