## 系统概述

### 什么是 SaaS 模式？

SaaS（Software as a Service，软件即服务）是一种通过互联网提供软件服务的模式。在 SaaS 模式下：

- ✅ **多租户架构**: 多个公司/组织共享同一套系统，但数据完全隔离
- ✅ **按需付费**: 根据使用量和功能选择合适的订阅计划
- ✅ **自动升级**: 系统功能自动更新，无需手动维护
- ✅ **随时随地**: 通过浏览器即可访问，无需安装软件

### 租户隔离机制

本系统采用**数据库级租户隔离**：

```
┌─────────────────────────────────────┐
│          系统数据库                   │
├─────────────────────────────────────┤
│  租户A (company-a)  │  租户B (company-b)  │
│  ├─ 用户数据     │  ├─ 用户数据     │
│  ├─ 物料数据     │  ├─ 物料数据     │
│  ├─ 订单数据     │  ├─ 订单数据     │
│  └─ ...          │  └─ ...          │
└─────────────────────────────────────┘
```

每个租户的数据通过 `tenantId` 字段严格隔离，确保数据安全。

## 注册与登录

### 🆕 新用户注册

**适用人群**: 首次使用系统的公司或组织管理员

#### 注册流程

1. **访问登录页面**: 打开系统登录页面
2. **切换到注册标签**: 点击"注册"标签
3. **填写注册信息**:
   - **公司名称**: 您的公司或组织名称（如：ABC科技有限公司）
   - **租户标识**: 用于唯一标识您的租户（如：abc-tech）
   - **管理员姓名**: 主要联系人的姓名
   - **邮箱地址**: 管理员邮箱（用于接收通知）
   - **密码**: 登录密码（至少6位）
   - **确认密码**: 再次输入密码

4. **阅读并同意条款**: 勾选"我已阅读并同意服务条款和隐私政策"
5. **点击注册**: 系统将自动创建租户和管理员账号

#### 注册完成后

- 系统会自动为您的公司创建以下内容：
  - ✅ 租户账号（试用版）
  - ✅ 管理员账号
  - ✅ 默认仓库
  - ✅ 默认账户
  - ✅ 编码规则
  - ✅ 示例数据

- 自动跳转到登录页面
- 填写注册信息会自动填充
- 直接登录即可使用

### 🔐 用户登录

#### 普通用户登录

1. **输入邮箱**: 系统账号或租户用户邮箱
2. **输入密码**: 对应的登录密码
3. **（可选）输入租户标识**: 如果多个租户使用相同邮箱
4. **点击登录**: 成功后进入系统

#### 超级管理员登录

1. **输入系统管理员邮箱**: 如 `admin@erp2026.com`
2. **输入密码**: 系统管理员密码
3. **点击登录**: 可访问所有租户数据

#### 登录成功后

- 根据用户角色显示不同的菜单和权限
- 顶部显示当前租户名称
- 可访问对应模块和功能

## 数据隔离

### 🔒 数据隔离机制

系统通过以下机制确保数据隔离：

#### 1. 数据库层面

```sql
-- 所有业务表都包含 tenantId 字段
CREATE TABLE suppliers (
  id VARCHAR(36) PRIMARY KEY,
  tenant_id VARCHAR(36) NOT NULL,
  name VARCHAR(100),
  ...
  INDEX idx_tenant_id (tenant_id)
);
```

#### 2. 应用层面

```typescript
// 所有数据库查询都自动添加租户过滤条件
const suppliers = await prisma.supplier.findMany({
  where: {
    tenantId: req.user.tenantId  // 自动添加租户过滤
  }
});
```

#### 3. API层面

```typescript
// 认证中间件自动验证租户权限
router.get('/suppliers', 
  authenticate,       // 检查登录
  tenantIsolation,   // 检查租户权限
  getSuppliers
);
```

### 🌐 不同公司的数据示例

#### 公司A (abc-tech)

```
供应商数据:
- 供应商1: 北京科技有限公司
- 供应商2: 上海贸易公司

物料数据:
- 物料A: 电阻 10K
- 物料B: 电容 100uF
```

#### 公司B (xyz-company)

```
供应商数据:
- 供应商1: 广州电子厂
- 供应商2: 深圳配件公司

物料数据:
- 物料A: 螺丝 M6
- 物料B: 螺母 M6
```

**重要**: 两家公司完全独立，数据互不影响！

### 🔄 数据访问流程

```
用户登录 → 获取Token → 请求API 
  ↓
系统验证Token → 提取tenantId → 查询数据
  ↓
查询时自动添加 WHERE tenantId = ?
  ↓
返回对应租户的数据
```

## 常见问题

### ❓ 通用问题

#### Q1: 如何确定我的租户标识？

**A**: 租户标识在注册时由系统根据公司名称自动生成，也可以自定义。
- 要求：只能包含小写字母、数字和连字符
- 唯一性：在整个系统中唯一
- 不可修改：注册后不可更改

#### Q2: 忘记租户标识怎么办？

**A**: 联系系统管理员或使用管理员邮箱登录查看。

#### Q3: 可以同时登录多个租户吗？

**A**: 不可以。一个账号只能属于一个租户。如果需要管理多个租户，需要每个租户单独的账号。

#### Q4: 数据安全吗？

**A**: 是的，我们采用多层安全措施：
- 数据库级隔离
- 传输加密 (HTTPS)
- 密码加密存储 (bcrypt)
- 定期数据备份
- 严格的访问控制

#### Q5: 如何升级或降级订阅？

**A**: 
1. 进入 **SaaS配置** → **套餐与账单**
2. 点击 **升级/降级**
3. 选择新计划
4. 确认并支付
5. 立即生效

#### Q6: 取消订阅后数据会丢失吗？

**A**: 不会。取消订阅后：
- 数据保留30天供导出
- 超过30天后自动删除
- 可随时重新订阅恢复数据

#### Q7: 如何添加用户？

**A**: 租户管理员可以添加用户：
1. 进入 **系统管理** → **用户管理**
2. 点击 **新增用户**
3. 填写用户信息
4. 设置角色和权限
5. 创建成功

#### Q8: 用户角色可以修改吗？

**A**: 可以，租户管理员可以修改普通用户的角色，但不能修改为管理员级别角色。

#### Q9: 如何备份数据？

**A**: 
1. 进入 **SaaS配置** → **安全设置**
2. 点击 **立即备份**
3. 备份完成后可下载
4. 建议定期备份

#### Q10: API如何使用？

**A**: 
1. 进入 **SaaS配置** → **API密钥**
2. 创建API密钥
3. 获取密钥后可在应用中调用API
4. 详细文档在 **集成与文档**中查看

### 🔧 技术问题

#### Q11: 登录时提示"租户不存在"？

**A**: 检查以下几点：
- 租户标识是否输入正确
- 租户是否已过期或停用
- 邮箱是否属于该租户

#### Q12: 登录后看不到某些菜单？

**A**: 检查用户角色权限：
- 菜单显示与用户角色相关
- 部分功能需要管理员权限
- 某些功能需要特定订阅计划

#### Q13: 数据显示错误或异常？

**A**: 请按以下步骤操作：
1. 清除浏览器缓存
2. 重新登录
3. 联系技术支持

#### Q14: 如何联系技术支持？

**A**: 
- 邮箱: support@erp2026.com
- 文档: https://docs.erp2026.com
- 在线客服: 系统右下角
