# 系统管理员租户管理指南

## 🎯 功能概述

系统管理员（super_admin）可以查看和管理所有租户，包括创建、编辑、删除、监控和管理租户的订阅、配额、功能等。

## 🔐 权限要求

**必需角色**: `super_admin` (系统管理员)

只有系统管理员才能访问租户管理功能，普通租户管理员无法看到相关菜单和功能。

## 📋 访问路径

### 方式一：通过菜单导航
1. 登录系统（使用系统管理员账号）
2. 在左侧菜单中找到 **"SaaS配置"** 分类
3. 点击 **"租户管理"** 进入租户管理页面

### 方式二：直接访问
在浏览器地址栏输入：`http://your-domain/tenants`

## 🎨 界面结构

租户管理页面包含以下主要功能区域：

### 1. 操作区域
- **新增租户按钮**: 创建新的租户
- **搜索框**: 搜索租户名称或标识
- **状态筛选**: 按租户状态筛选（启用/停用/试用中/已过期）
- **刷新按钮**: 刷新租户列表

### 2. 租户列表表格
显示所有租户的详细信息，包括：
- **租户名称**: 租户的显示名称
- **租户标识**: 系统内唯一标识
- **计划类型**: 订阅计划（免费/基础/专业/企业）
- **状态**: 当前状态（启用/停用/试用中/已过期）
- **用户数**: 当前用户数/配额用户数
- **存储空间**: 已用存储/总存储配额
- **创建时间**: 租户创建日期
- **操作**: 查看详情、编辑、删除等操作

## 🔧 核心功能

### 1. 查看租户列表

#### 功能说明
- 默认显示所有租户的列表
- 支持分页查看
- 支持按名称、标识搜索
- 支持按状态筛选
- 支持按创建时间、名称排序

#### 操作步骤
1. 进入租户管理页面
2. 查看默认显示的租户列表
3. 使用搜索框查找特定租户
4. 使用状态筛选器过滤租户

#### 示例
```
搜索: "ABC科技" 或 "abc-tech"
状态: 选择 "试用中" 查看所有试用租户
```

### 2. 创建新租户

#### 功能说明
系统管理员可以手动创建新的租户，为其分配管理员账号和初始配置。

#### 操作步骤
1. 点击 **"新增租户"** 按钮
2. 填写租户基本信息：
   - **公司名称**: 租户的公司名称（必填，如：ABC科技有限公司）
   - **租户标识**: 系统唯一标识（必填，如：abc-tech）
   - **显示名称**: 简短显示名（可选）
   - **联系邮箱**: 管理员邮箱（必填）
   - **联系电话**: 公司电话（可选）
   - **地址**: 公司地址（可选）
   - **税号**: 公司税号（可选）
   - **网站**: 公司网站（可选）
3. 配置管理员信息：
   - **管理员姓名**: 初始管理员姓名（必填）
   - **管理员邮箱**: 初始管理员邮箱（必填）
   - **管理员密码**: 初始管理员密码（必填）
4. 选择订阅计划：
   - **计划类型**: 免费版/基础版/专业版/企业版
   - **计费周期**: 月付/年付/2年付/3年付
5. 配置功能开关：
   - 多仓库、多币种、自定义字段、API访问等
6. 点击 **"确认创建"**

#### 注意事项
- 租户标识必须在系统中唯一
- 租户名称不能重复
- 管理员邮箱不能与现有用户重复
- 创建后自动生成初始数据（默认仓库、账户、编码规则等）

### 3. 查看租户详情

#### 功能说明
查看租户的详细配置、使用情况、订阅信息等。

#### 操作步骤
1. 在租户列表中找到目标租户
2. 点击操作列的 **"查看详情"** 按钮
3. 查看详情面板，包含：

**基本信息**:
- 公司名称、标识、显示名称
- 联系信息（邮箱、电话、地址、税号、网站）
- 时区、语言、货币设置

**订阅信息**:
- 当前订阅计划
- 计费周期和费用
- 试用期结束时间
- 计费周期起止时间

**功能配置**:
- 功能开关状态（多仓库、多币种、API等）
- 配额设置（用户数、存储空间、API调用量）

**使用统计**:
- 用户使用情况（已用/配额）
- 存储使用情况（已用/配额）
- API调用情况（本月/配额）

**租户管理员信息**:
- 管理员姓名、邮箱、角色
- 最后登录时间

### 4. 编辑租户信息

#### 功能说明
修改租户的基本信息、联系信息、功能配置等。

#### 操作步骤
1. 在租户列表中找到目标租户
2. 点击操作列的 **"编辑"** 按钮
3. 修改需要更新的信息：
   - 基本信息（公司名称、标识等）
   - 联系信息（邮箱、电话、地址等）
   - 功能开关配置
   - 配额设置
4. 点击 **"保存更改"**

#### 限制说明
- 租户标识创建后不可修改
- 某些核心配置修改可能需要租户重新登录
- 降低配额可能影响租户正常使用

### 5. 管理租户状态

#### 功能说明
控制租户的启用/停用状态，处理过期租户。

#### 操作步骤
1. 在租户列表中找到目标租户
2. 点击操作列的 **"更多"** → **"启用/停用"**
3. 确认操作

#### 状态说明
- **启用 (active)**: 租户正常使用所有功能
- **停用 (suspended)**: 租户被暂停，无法访问系统
- **试用中 (trial)**: 租户处于试用期
- **已过期 (expired)**: 租户订阅已过期
- **已取消 (cancelled)**: 租户已主动取消订阅

### 6. 删除租户

#### 功能说明
彻底删除租户及其所有数据（不可恢复）。

#### 操作步骤
1. 在租户列表中找到目标租户
2. 点击操作列的 **"删除"** 按钮
3. 确认删除操作

#### ⚠️ 重要警告
- **删除操作不可撤销**
- 将删除租户的所有数据（用户、订单、库存等）
- 建议删除前先备份数据
- 如有未结订单或欠款，需先处理完毕

### 7. 查看租户用户

#### 功能说明
查看租户下的所有用户及其角色、状态。

#### 操作步骤
1. 在租户详情面板中找到 **"租户用户"** 标签
2. 查看用户列表，包括：
   - 用户姓名、邮箱、角色
   - 用户状态（启用/停用）
   - 部门、职位信息
   - 最后登录时间

### 8. 管理订阅和账单

#### 功能说明
为租户升级/降级订阅计划，管理账单信息。

#### 操作步骤
1. 在租户详情面板中找到 **"订阅管理"** 标签
2. 查看当前订阅信息
3. 点击 **"变更计划"** 修改订阅
4. 查看历史账单记录

#### 可操作功能
- 升级/降级订阅计划
- 修改计费周期
- 查看账单历史
- 下载发票PDF

### 9. 查看审计日志

#### 功能说明
查看租户的操作日志和系统事件。

#### 操作步骤
1. 在租户详情面板中找到 **"审计日志"** 标签
2. 按时间、操作类型筛选
3. 查看具体操作详情

#### 日志类型
- 用户登录/登出
- 数据创建/修改/删除
- 权限变更
- 系统配置修改
- API调用记录

## 📊 统计仪表盘

### 总体统计
在租户管理页面顶部显示：
- **总租户数**: 系统中租户总数
- **活跃租户数**: 当前启用的租户数
- **试用租户数**: 处于试用期的租户数
- **过期租户数**: 已过期的租户数
- **本月新增**: 本月新创建的租户数

### 使用趋势
- 租户增长趋势图
- 活跃度变化趋势
- 订阅收入统计

## 🎯 最佳实践

### 1. 租户命名规范
- 租户标识：小写字母、数字、连字符（如：abc-tech）
- 公司名称：正式公司全称
- 显示名称：简短易记的公司名

### 2. 定期监控
- 每周检查试用即将到期的租户
- 监控接近配额限制的租户
- 关注异常活跃或异常不活跃的租户

### 3. 数据安全
- 删除租户前必须备份数据
- 敏感操作需二次确认
- 定期审查租户权限配置

### 4. 客户支持
- 为试用租户提供培训和支持
- 及时响应租户问题和需求
- 定期收集租户反馈

## 🔍 常见问题

### Q1: 如何查看某个租户的使用情况？
A: 点击租户列表中的"查看详情"，在详情面板中可以看到该租户的使用统计数据。

### Q2: 可以为租户临时增加配额吗？
A: 可以，在租户详情中修改配额设置，系统会记录调整历史。

### Q3: 租户无法登录怎么办？
A: 首先检查租户状态是否为"启用"，然后检查管理员账号是否正常，必要时可以重置密码。

### Q4: 如何处理过期租户？
A: 过期租户会自动变为"已过期"状态，可以联系续费或删除。

### Q5: 系统管理员能看到租户的具体业务数据吗？
A: 系统管理员只能看到租户的配置和统计信息，不能查看具体的业务数据（如订单、库存等），需要切换到该租户才能查看。

### Q6: 如何批量导出租户列表？
A: 在租户管理页面可以导出Excel格式的租户列表。

## 📱 快捷键

- **F5**: 刷新租户列表
- **Ctrl + F**: 聚焦搜索框
- **Esc**: 关闭详情面板
