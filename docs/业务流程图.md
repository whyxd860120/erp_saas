### 2️⃣ 采购管理模块

#### 2.1 采购订单
**验证点：**
- [ ] 新增采购订单
- [ ] 选择供应商
- [ ] 添加商品明细（数量、单价）
- [ ] 预计到货日期
- [ ] 提交审批（如果启用了审批）
- [ ] 查看审批状态
- [ ] 查看历史价格
- [ ] 订单关闭

**流程：**
```
创建采购订单 → 选择供应商 → 添加明细 → 设置到货日期 → 提交 → 审批流程 → 确认
  ↓
审批状态：pending → approved → completed
```

#### 2.2 采购入库单
**验证点：**
- [ ] 从采购订单生成入库单
- [ ] 手动创建入库单
- [ ] 选择仓库
- [ ] 红字入库单（退货）
- [ ] 蓝字入库单（正常入库）
- [ ] 填写批次号（如果商品启用批次管理）
- [ ] 填写生产日期/过期日期
- [ ] 质检结果（合格/不合格）
- [ ] 确认入库（实际扣减库存）

**流程：**
```
创建入库单 → 选择来源（采购订单/手动）→ 选择仓库 → 红蓝字选择 → 
填写明细 → 批次信息 → 质检 → 确认入库 → 库存增加
```

#### 2.3 供应商绩效

## 🧪 验证步骤建议

### 第一阶段：基础验证
1. 登录系统 → 进入用户管理 → 创建测试用户
2. 进入角色管理 → 创建测试角色 → 分配功能权限
3. 进入商品管理 → 创建测试商品（启用批次/保质期/SN）

### 第二阶段：采购验证
1. 创建供应商
2. 创建采购订单 → 提交审批 → 确认
3. 创建采购入库单 → 选择批次 → 确认入库

4. 创建收款单 → 核销测试

2. 配置数据权限（只能看自己的单据）

### 第五阶段：高级功能验证
1. 创建库存盘点单 → 盘盈/盘亏 → 确认调整
2. 查看保质期预警 → 过期商品

## 🎯 关键测试点

1. **数据权限验证** - 最核心功能
   - 创建角色A：可以查看所有销售订单
   - 创建角色B：只能查看自己的销售订单
   - 创建用户1（角色A）和用户2（角色B）
   - 分别登录，验证数据隔离

2. **红蓝字验证**
   - 创建蓝字入库单 → 库存增加
   - 创建红字入库单 → 库存减少
   - 创建蓝字出库单 → 库存减少
   - 创建红字出库单 → 库存增加

3. **批次管理验证**
   - 创建商品（启用批次管理）
   - 入库时填写批次号
   - 出库时选择批次
   - 查看批次库存

4. **保质期验证**
   - 创建商品（启用保质期，设置预警天数）
   - 入库时填写生产日期/过期日期
   - 查看保质期预警列表

5. **SN码验证**
   - 创建商品（启用SN码）
   - 批量生成SN码
   - 出库时扫描SN码
   - 查询SN码状态
